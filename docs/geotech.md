# Workflow

1. Leapfrogmodel
2. Export to CAD (.dwg or .dxf)
3. Open Ground Model CAD in Rhino
   1. Send to Speckle
   2. Export to [`.usd`](../Geotech/LeapFrogGroundModel.usd), [`.usda`](../Geotech/LeapFrogGroundModel.usda), [`.usdz`](../Geotech/LeapFrogGroundModel.usdz)
